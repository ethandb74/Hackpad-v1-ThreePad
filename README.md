# Hackpad-v1-ThreePad

ThreePad is a three key HackPad :)

Building a three key hackpad for various computer science or EE purposes made to improve workflow and productivity. 
Main use is for KiCAD Shortcuts in PCB design ex. -- Add Component (A), Add Wire (W), and Add GND Symbol (P)

## Hardware
* **Controller:** Seeed XIAO RP2040
* **Pins Used:** D9, D10, D11 (Direct to GND)
* **Firmware:** KMK -Pyton
